安裝模組：
　　　1. 點 [ CivitAI ]
　　　2. 打勾 [ Search by term? ]
　　　3. [ Srearch term ] 欄位輸入你想搜尋的模型
　　　4. 按下 [ Get List ] －稍微等兩秒－
　　　5. 選擇想要的模型
　　　6. 選擇模型的版本
　　　7. 選擇模型的檔案名稱
　　　8. 點下載
